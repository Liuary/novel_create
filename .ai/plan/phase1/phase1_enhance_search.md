# 强化一期 — 搜索与替换功能

## 状态

✅ 已完成 (2026-05-01)

## 目标

在内容管理侧边栏和章节编辑器中加入全局搜索、章节内搜索/替换功能，提升长篇创作的内容定位效率。

---

## 1. 左侧全局搜索栏

### 位置
侧边栏内容区域下方、Tab 切换上方，仅"内容树"Tab 显示。

### 交互
- 输入关键词，按回车或点击🔍按钮开始搜索
- 搜索结果以覆盖层替换内容树显示
- 搜索框内 × 按钮清除搜索，恢复内容树

### 搜索结果
每条结果两行：
- **第一行**：卷名 / 章节名（关键词高亮）
- **第二行**：正文预览（关键词第一次出现的上下文，前后各30字符，超出…）

如果正文不含关键词，仅显示第一行。

### 排序
- 当有打开章节时，按与当前章节的距离排序（同书内按章节先后位置）
- 无打开章节时，按第一个卷的第一个章节开始排列

### 点击行为
- **点击第一行** → 打开对应章节
- **点击第二行** → 打开对应章节并自动激活章节内搜索（定位到匹配位置）

### 空状态
"未查找到相关内容"

---

## 2. 编辑器内联搜索

### 位置
章节编辑器标题栏上方浮动显示，替代原有自动保存提示区域。

### 入口
标题栏右侧搜索按钮 🔍（写作/阅读模式按钮旁，点击切换显示/隐藏）

### 搜索栏
- 单行输入框，输入后回车或点击搜索按钮开始搜索
- 搜索框弹出菜单可切换"搜索/替换"模式
- 搜索完成显示左侧 < 按钮、匹配计数（当前/总数）、> 按钮
- 0结果时计数显示为红色

### 导航
- `<` 切换到上一个匹配
- `>` 切换到下一个匹配
- 切换时自动滚动到匹配位置并选中文本

### 替换模式
- 搜索框下方出现第二行"替换为..."输入框
- 导航工具栏右侧显示"替换"按钮，点击替换当前匹配
- 工具栏左侧显示"替换全部"按钮
- 搜索框弹出菜单可切换回纯搜索模式

### 导航栏按钮
| 按钮 | 功能 |
|------|------|
| `<` | 上一个匹配（0结果时置灰） |
| `1/5` | 当前/总数（0结果时红色） |
| `>` | 下一个匹配（0结果时置灰） |
| 替换全部 | 替换所有匹配 |
| 替换 | 替换当前选中匹配（替换模式） |

---

## 3. 搜索/替换模式切换

- 左侧搜索栏和编辑区内联搜索均支持下拉菜单切换"搜索"/"替换"
- 替换模式下：双输入栏（查找 + 替换为），双按钮（查找 + 替换全部）
- 编辑区内联搜索替换模式额外提供"替换"按钮（仅替换当前匹配）
