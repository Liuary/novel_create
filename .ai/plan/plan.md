# 小说创作工具 - 开发计划

## 项目概述

基于 Flutter 的 PC 端小说创作工具，提供书籍章节管理、辅助创作工具集、AI 辅助创作流程。

## 技术选型

| 项目 | 选择 |
|------|------|
| 数据存储 | JSON 文件（正文）+ SQLite/Drift（元数据/关联） |
| 状态管理 | Riverpod |
| 编辑器 | 纯文本 TextField（写作/阅读双模式） |
| AI 接入 | OpenAI 兼容 API |

## 阶段规划

- **阶段1**: 基础框架 + 书籍章节编辑 ✅ 已完成
- **阶段2**: 模块化知识库系统（角色、大纲、背景、灵感、物品）🔄 进行中
- **阶段3**: AI 协作写作引擎（多AI并行创作、交叉审阅、自动知识库更新）📋 展望计划已制定
