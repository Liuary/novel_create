# 小说创作工具 - 开发计划

## 项目概述

基于 Flutter 的 PC 端小说创作工具，提供书籍章节管理、辅助创作工具集、AI 辅助创作流程。

## 技术选型

| 项目 | 选择 |
|------|------|
| 数据存储 | JSON 文件（用户文档目录） |
| 状态管理 | Riverpod |
| 富文本编辑 | flutter_quill |
| AI 接入 | OpenAI 兼容 API |

## 阶段规划

- **阶段1**: 基础框架 + 书籍章节编辑（当前）
- **阶段2**: 辅助创作工具（角色、大纲、时间线、剧情分支等）
- **阶段3**: AI 辅助创作（主线生成、风格管理、滑动创作、审阅流程）
