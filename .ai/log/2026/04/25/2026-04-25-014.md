# 2026-04-25-014 - 章节切换双弹窗/倒计时消失/阅读模式布局修复 + 全局消息系统 + 侧边栏两级重构

## 背景

`phase1-polish` 分支继续打磨修复。

## 改动

### bug 修复
- **双弹窗**: `_onLeavingChapter()` 取消时调用 `_restoreProviderState()` 恢复三个 provider；guard 延迟清理（`Future.microtask`）
- **右下角倒计时消失**: 修复 timer 覆盖问题
- **重启后章节不加载**: `_onChapterChanging` else 分支加 `_loadCurrentChapter()`
- **阅读模式布局错乱**: Scrollbar 移入 CustomPaint child 内直接包裹 TextField；移除 `StrutStyle(forceStrutHeight: true)`
- **`ref.listen` unmount 崩溃**: 三个 listen 回调 + `_loadCurrentChapter` + 异步回调加 `if (!mounted) return`；`dispose()` 中 `ref.read` 移到 `deactivate()`
- **文字改回存档版本时倒计时清空**
- **Riverpod 3.x API 适配**: `StateNotifier` → `Notifier`；`StateProvider` 从 legacy.dart import
- **编辑器花括号不匹配编译错误**

### 新功能
- **全局消息提示系统**: `ToastService`（消息队列）+ `ToastOverlay`（右下角 UI），各模块 `ref.read(toastProvider.notifier).show(...)` 调用
- **侧边栏两级重构**: 选中书籍后标题栏显示书籍名称（可点击返回），下方只显示卷和章
- **自动保存倒计时 UI 调整为左下角纯文字+描边**，全局消息置于右下角
- **倒计时结束不显示"自动保存中…"，直接消失**

### 文件变动
- 新增: `lib/services/toast_service.dart`、`lib/widgets/toast_overlay.dart`、`lib/models/user_config.dart`
- 重写: `lib/widgets/editor_page.dart`、`lib/widgets/sidebar.dart`
- 修改: `lib/pages/home_page.dart`、`lib/providers/app_providers.dart`、`lib/services/storage_service.dart`
